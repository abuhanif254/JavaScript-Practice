# JavaScript Practice

Welcome to the **JavaScript Practice** repository! This project contains a collection of JavaScript files designed to help you learn, practice, and master core JavaScript concepts through hands-on coding examples.

## 📂 Folder Structure

- `advance_javascript.js` — Advanced JavaScript concepts and examples
- `array_practice.js` — Array methods and practice problems
- `arrow_function.js` — Arrow function syntax and use cases
- `csj_array.js` — Custom array-related exercises
- `example_logical_operators..js` — Logical operators in JavaScript
- `function.js` — Function declarations, expressions, and usage
- `iterating_arrays.js` — Techniques for iterating over arrays
- `js_array.js` — JavaScript array basics and operations
- `loop.js` — Looping constructs (for, while, etc.)
- `object.js` — Object creation and manipulation
- `practical_example.js` — Real-world JavaScript examples
- `practical_example2.js` — Additional practical JavaScript examples

## 🚀 Getting Started

1. **Clone the repository:**
   ```sh
   git clone https://github.com/abuhanif254/JavaScript-Practice.git
   ```
2. **Open in your favorite code editor (e.g., VS Code).**
3. **Explore each file and run the code snippets in your browser console or Node.js.**
